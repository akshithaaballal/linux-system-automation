\# Windows System Script



Just a simple PowerShell script I made to check basic system stuff in one go.



\## What it does

\- Shows system info

\- Checks disk + memory

\- Lists top processes

\- Displays open ports



\## How to run



Set-ExecutionPolicy RemoteSigned -Scope CurrentUser



.\\system\_maintenance.ps1



\## Why I made this

Trying to get more comfortable with PowerShell and system-level stuff.
